## 5. SUS SO'ROVNOMA ALGORITMI

```
SUS Ball = ОЈ(savollar) Г— 2.5

Har bir savol uchun:
- Ijobiy savollar (1,3,5,7,9): ball = javob - 1
- Salbiy savollar (2,4,6,8,10): ball = 5 - javob

Minimum: 0  |  Maksimum: 100

Talqin:
- 85-100: A'lo (Excellent)
- 72-84:  Yaxshi (Good)
- 52-71:  O'rta (OK)
- 38-51:  Qoniqarli (Poor)
- 0-37:   Yomon (Awful)
```

## 6. JAVOB VAQTI HISOBLASH

```javascript
// O'qituvchining javob berish vaqti
const calcResponseTime = (messages, teacherId) => {
  const studentMsgs = messages.filter(m => m.senderId !== teacherId);
  const teacherReplies = messages.filter(m => m.senderId === teacherId);
  
  let totalTime = 0;
  let count = 0;
  
  studentMsgs.forEach(studentMsg => {
    const nextTeacherMsg = teacherReplies.find(tm => 
      tm.createdAt.seconds > studentMsg.createdAt.seconds
    );
    if (nextTeacherMsg) {
      totalTime += (nextTeacherMsg.createdAt.seconds - studentMsg.createdAt.seconds) / 60;
      count++;
    }
  });
  
  return count > 0 ? (totalTime / count).toFixed(1) : 0; // daqiqada
};
```

## 7. FAOLLIK-NATIJA KORRELYATSIYASI

```
Pearson korrelyatsiyasi formulasi:
r = ОЈ(xi - xМ„)(yi - Иі) / вҲҡ[ОЈ(xi - xМ„)ВІ Г— ОЈ(yi - Иі)ВІ]

Bu yerda:
x = talaba chat faolligi (xabarlar soni)
y = test natijalari (0-100)

Ilmiy taxmin: r вүҘ 0.6 bo'lsa вҖ” kuchli ijobiy korrelyatsiya
```

## 8. XAVFSIZLIK CHORALARI

1. **Firebase Auth** вҖ” JWT asosidagi autentifikatsiya
2. **Firestore Rules** вҖ” Server tomonidan ruxsat tekshirish
3. **MIME tekshirish** вҖ” Fayl yuklashda tur nazorati
4. **Fayl hajmi chegarasi** вҖ” 10MB maksimum
5. **Rate limiting** вҖ” 10 xabar / 10 soniya (frontend)
6. **isMuted flag** вҖ” Foydalanuvchini cheklash imkoniyati

## 9. PILOT SINOV REJASI

### Qatnashchilar
- 10 ta o'qituvchi
- 30 ta talaba (3 kurs Г— 10 talaba)

### Sinov Bosqichlari
1. **Onboarding** (1 hafta) вҖ” Tizimni o'rgatish
2. **Faol ishlatish** (2 hafta) вҖ” Dars jarayonida qo'llash
3. **Baholash** (1 hafta) вҖ” SUS so'rovnoma + intervyu

### O'lchanadigan ko'rsatkichlar
- O'rtacha SUS bali (maqsad: вүҘ 72)
- O'qituvchi javob vaqti (maqsad: вүӨ 5 daqiqa)
- Kunlik faol foydalanuvchilar
- Xabarlar soni / kurs
